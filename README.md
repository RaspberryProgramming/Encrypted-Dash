# Encrypted-Dash
This programmed is designed to let you create a dash cam that encrypts the recorded video.

HOW-TO:

cd Encrypted\ Dash

pip3 install -r requirements.txt

python3 genkeys.py

python3 camera.py

python3 decryptVideo.py
